# twitter-repos-bot

:bird: Twitter bot to fetch tweets with GitHub repositories written with Node.js

## Build Setup

``` bash
# clone repository
git clone https://github.com/lexmartinez/twitter-repos-bot.git

# open project folder
cd twitter-repos-bot

# install dependencies
npm install

# run app
npm start
```
